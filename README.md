# 📘 Assignment-01: HTML Learning Demo Page (TuteDude)

## 🚀 Project Overview
This project is a **comprehensive HTML demo page** created for a fictional learning platform called **TuteDude**.  
It demonstrates the use of semantic HTML elements, multimedia embedding, forms, tables, and accessibility features.

---

## 📑 Page Structure
- **Header**: Welcoming heading, description, navigation menu, hero image with caption.  
- **Main**: Contains About, Menu, Trivia, and Extras sections.  
- **Aside**: Trivia facts with `<details>` and `<summary>`.  
- **Footer**: Contact details and copyright.

---

## 🏗️ Features Implemented
- Semantic HTML tags (`<header>`, `<main>`, `<section>`, `<footer>`, etc.)
- Highlighted text using `<mark>`, `<em>`, `<strong>`
- Navigation menu with `<nav>`
- Hero image with `<figure>` and `<figcaption>`
- About section with `<article>`, `<abbr>`, `<time>`, `<blockquote>`, `<cite>`
- Menu section with complex tables and nested tables
- Trivia section with `<aside>`, `<details>`, `<summary>`
- Extras section with `<audio>` and `<video>`
- Feedback form with `<fieldset>`, `<legend>`, inputs, dropdowns, textarea, and submit button
- Footer with `<address>` and `<time>`
